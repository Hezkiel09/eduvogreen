# Tech Stack EduVoGreen Project

## 🎯 **Overview**
EduVoGreen adalah aplikasi mobile cross-platform yang dibangun menggunakan **Flutter** untuk pengelolaan konten edukasi, volunteer events, dan admin dashboard. Aplikasi ini mendukung fitur authentication, article management, volunteer reporting, dan multi-role access (user, admin).

## 💡 **Key Features Implemented**
- **Auth**: Email/password signup/login, reset password via Supabase Auth
- **Admin**: Dashboard, Tambah/Tinjau/Revisi Artikel, Unggah Volunteer
- **Volunteer**: Event listing, detail, reporting with image picker
- **EduHub**: Article management with custom service/cubit
- **Responsive UI**: Bottom nav, tabs, popups, cards

Tech stack ini scalable, modern, dan cost-effective berkat Supabase (no backend code needed). Ready for production dengan minor tweaks seperti env vars untuk Supabase keys!
